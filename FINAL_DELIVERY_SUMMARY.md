## 🚀 COMO USAR

### 1. Setup Inicial

```bash
# Clone o projeto
cd estudio_ia_videos

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Edit .env.local com suas credenciais

# Run migrations
npx prisma migrate dev

# Start development server
npm run dev
```

### 2. Criar um Vídeo

```bash
# Via UI
1. Acesse http://localhost:3000/studio
2. Upload PPTX ou crie slides manualmente
3. Selecione avatar e voz
4. Clique em "Generate Video"
5. Aguarde rendering (30s - 2min dependendo do tier)
6. Download ou compartilhe

# Via API
curl -X POST http://localhost:3000/api/v2/avatars/render \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Olá, bem-vindo ao curso",
    "avatarId": "default",
    "quality": "STANDARD",
    "voice": "pt-BR-FranciscaNeural"
  }'
```

### 3. Deploy em Produção

```bash
# Configure production environment
cp .env.example .env.production
# Edit .env.production

# Deploy to Vercel
vercel --prod

# Run post-deployment checks
curl https://your-domain.com/api/health
```
