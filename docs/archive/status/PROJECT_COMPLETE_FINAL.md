## 🚀 DEPLOYMENT

```bash
# Quick start
git clone <repo>
cd estudio_ia_videos
npm install
cp .env.example .env.local
# Configure API keys in .env.local
docker-compose up -d
npx prisma migrate deploy
npm run build
npm start
```

**Required API Keys**:
- DATABASE_URL, REDIS_URL
- AZURE_SPEECH_KEY (lip-sync)
- ELEVENLABS_API_KEY (voice cloning)
