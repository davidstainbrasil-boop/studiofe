# Imagens dos Avatares

Este diretório deve conter as imagens dos avatares utilizadas no passo de escolha de apresentador.

## Arquivos esperados:
- ana.jpg
- carlos.jpg
- julia.jpg
- roberto.jpg
- sofia.jpg
- pedro.jpg

As imagens devem ter preferencialmente a proporção 3:4 (retrato).
Caso a imagem não seja encontrada, o sistema exibirá um ícone de fallback.
