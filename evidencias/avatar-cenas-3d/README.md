# Evidências — Avatar Cenas 3D (Execução REAL)

Este diretório é a fonte operacional de evidências para execução do plano:
- `docs/plano-implementacao-avatar-cenas-3d.md`

## Estrutura
- `fase-1/T1/` ... evidências da task
- `checkpoints/` ... handoffs por sessão

## Regra de ouro
Uma task só é concluída quando os artefatos mínimos existirem e forem reproduzíveis.

## Artefatos mínimos por task
1. `README.md` (resumo da execução)
2. `request-response.json` (entrada/saída principal)
3. `logs.txt` (log técnico relevante)
4. `metrics.json` (tempo/custo/status)
5. `artifact.txt` (URL/ID do artefato)

## Convenção de nome
`<data>-<task>-<tipo>-<status>`

Exemplos:
- `2026-02-14-T1-migration-ok`
- `2026-02-14-T5-render-completed`
