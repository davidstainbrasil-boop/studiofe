- Early return tardio (após operações que podem falhar)

   ```typescript

   ```
   - Exclusões explícitas: `_next/static`, `_next/image`, `_next/data`
   - Extensões: `.css`, `.js`, `.map`, `.svg`, etc.

   ```typescript
   if (pathname.startsWith('/_next/static') ||

     return NextResponse.next();  // Bypass IMEDIATO
   ```
